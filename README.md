## ğŸ”„ How Playlists Update Automatically

1. Add a new folder inside the `songs/` directory  
2. Add `.mp3` files and a cover image (`jpg / png / webp`)  
3. Commit and push changes to GitHub  
4. Backend redeploys automatically on Render  
5. Frontend fetches updated playlists without code changes
